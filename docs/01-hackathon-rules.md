## Hard rules (every submission must show)
1. **≤ $0.01 per action**
2. **≥ 50 on-chain transactions** in the demo
3. **Written margin analysis** — why would fail at normal gas
4. **Public GitHub repo, MIT license**
5. **Video** showing end-to-end USDC tx via Circle Developer Console + Arc Block Explorer verification
6. Explicit **track declaration** + **Circle products used**
7. **Circle Product Feedback** form (required)
