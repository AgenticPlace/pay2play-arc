Honest framing

pay2play is a thin composition over Circle's @circle-fin/x402-batching and @x402/mcp from the x402 monorepo. We do not reimplement settlement, the x402 spec, or batching.

What we add: a unified UsageSignal meter that normalizes every metering axis into one price function, a voucher Session for streaming flows (client accumulates, server flushes), and per-angle modular components that demonstrate the abstraction across all four hackathon tracks. Every prior winner rebuilt the same payment plumbing for one vertical; pay2play is the plumbing.

Quick start

git clone https://github.com/pay2play/pay2play.git
cd pay2play
pnpm install
cp .env.example .env            # fill SELLER_ADDRESS, BUYER_PRIVATE_KEY
pnpm generate-wallets           # or reuse existing
# Visit https://faucet.circle.com/  to fund buyer with testnet USDC
pnpm --filter c3-llm-stream start    # →  http://localhost:4023/
